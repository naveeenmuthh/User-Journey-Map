Actions
  • Prepares lesson plan in a separate doc
  • Uploads materials to LMS (Canvas / Google Classroom)
  • Sets up Zoom meeting separately
  • Shares meeting link via email or LMS announcement
  • Manually configures breakout rooms
Uses 3+ disconnected tools before class begins
  • Opens Zoom; admits students from waiting room
  • Verbally calls attendance in chat or aloud
  • Re-shares materials students can't find in LMS
  • Troubleshoots student mic/camera issues
First 10 min lost to admin before teaching begins
  • Delivers lesson via screen share or whiteboard
  • Monitors chat for questions while speaking
  • Manages breakout rooms manually
  • Switches between Zoom + LMS + whiteboard tabs
  • Tries to gauge who is engaged vs disengaged
Juggles 3 active tools simultaneously
  • Calls on students verbally; monitors reactions
  • Answers questions in chat while continuing lesson
  • Uses polls or breakout rooms for group work
  • Attempts to re-engage visibly passive students
No data on who's disengaged until it's too late
  • Manually compiles attendance from memory / chat log
  • Downloads Zoom recording; re-uploads to LMS
  • Sends recap email separately
  • Messages parents about absences individually
  • Plans next session across multiple tools again
30–45 min of post-class admin per session
